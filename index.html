Fitur

Engine Direct — Tanpa Browser
VeloDash v2 upload langsung lewat API Fotoyu, bukan otomasi Chrome. Hasilnya jauh lebih cepat, ringan di RAM, dan stabil — tanpa tab yang stuck atau nyangkut.
Auto-Fill + Video
Harga, FotoTree, deskripsi, dan tanggal diisi otomatis dengan smart matching. Dukung foto maupun video, plus preset yang bisa disimpan per event.
Dashboard Pendapatan

Analitik Penjualan

Windows & Mac

Panduan

Panduan lengkap — dari install sampai analitik

Ikuti langkahnya sekali, seterusnya tinggal ulang. Panduan yang sama juga ada di dalam aplikasi (tab Panduan).

1

Install & buka VeloDash

  • Download installer untuk Windows atau macOS, lalu jalankan seperti biasa.
  • Buka aplikasi. Kamu bisa langsung pilih Trial 1 hari gratis, atau masukkan serial number untuk aktivasi penuh.
2

Login akun Fotoyu

  • Buka tab Accounts → Tambah akun, isi username/email + password Fotoyu kamu.
  • Kalau akunmu pakai 2FA, masukkan kode OTP saat diminta.
  • Bisa simpan banyak akun sekaligus dan ganti-ganti lewat dropdown. Password tidak disimpan — hanya token login yang tersimpan aman di device.
3

Upload foto & video

  • Di tab Upload, pilih folder — file di sub-folder ikut ke-scan otomatis.
  • Pilih mode: Foto, Video, atau Foto + Video.
  • Isi Harga, FotoTree (ketik lalu pilih dari saran), dan Deskripsi. Bisa disimpan sebagai preset.
  • Klik Start Upload. File yang gagal diulang otomatis di akhir batch.
4

Atur kecepatan sesuai koneksi

  • Di tab Settings, pilih preset: Hemat (internet lambat / sering timeout), Seimbang (default), atau Ngebut (internet kencang & CPU kuat).
  • Kalau upload sering timeout, turunkan preset ke Hemat.
5

Pantau penjualan & riset

  • Dashboard: ringkasan pendapatan & transaksi terbaru. Analytics: tren penjualan dari waktu ke waktu.
  • Tree Analyzer: scan FotoTree untuk lihat ranking foto terjual per seller — bahan riset event & harga.
  • Di masa Trial, preview data dibatasi 7 hari terakhir; rentang penuh terbuka setelah upgrade.
Masih bingung atau butuh reset aktivasi saat pindah device? Chat admin via WhatsApp — dibantu langsung.
