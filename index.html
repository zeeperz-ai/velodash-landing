Multi-Tab Paralel
Upload hingga 20 tab secara bersamaan dalam 1 Chrome window — batasnya bisa kamu atur sendiri biar RAM device tetap aman. Ribuan foto terupload dalam hitungan menit.
🤖
Auto-Fill Form
Harga, FotoTree, deskripsi, dan tanggal diisi otomatis. FotoTree dipilih dengan smart matching — exact & subset.
🔁
Resume Foto Gagal
Foto yang gagal dicatat ke failed.txt. Klik Resume Failed untuk mengulang hanya foto yang belum sukses.
🔑
Session Tersimpan
Login sekali, diingat selamanya. Tidak perlu input password setiap sesi — aman tersimpan di device kamu.
📊
Laporan Upload
Setiap sesi menghasilkan laporan lengkap: berhasil, duplikat, gagal, kecepatan, durasi — export ke file TXT.
💻
Windows & Mac
Satu lisensi, dua platform. Build native untuk Windows (.exe) dan macOS Apple Silicon (.app).

FAQ

Pertanyaan yang sering ditanyakan

Ya. VeloDash menggunakan browser Chrome yang sama seperti kamu buka manual — tidak ada injection API yang tidak resmi. Upload dilakukan persis seperti user biasa mengklik tombol. Session login disimpan aman di device kamu, tidak dikirim ke server manapun.
Maks. 2000 foto atau 50 video per batch (mengikuti limit Fotoyu). Dengan PRO, batch ini bisa dijalankan di banyak tab sekaligus (hingga 20, disesuaikan dengan RAM device kamu) — jadi total per putaran bisa jauh lebih besar dari 1 batch.
1 lisensi = 1 perangkat aktif. Mau pindah ke device baru? Klik "Lepaskan Device Ini" di aplikasi lama, lalu aktivasi ulang di device baru — bisa dilakukan sendiri kapan saja. Kalau device lama sudah tidak bisa diakses, hubungi support untuk reset manual.
Tergantung RAM device dan kecepatan internet kamu. VeloDash punya pengaturan "Batas Tab Aman" yang bisa kamu turunkan sendiri kalau upload mulai macet/stuck di koneksi lambat — jadi kamu yang kontrol keseimbangan kecepatan vs stabilitas.
